- Если в Кеше хранится полный набор данных, то:
  - Применить фильтр;
  - Применить сортировку;
  - Разбить на страницы.
- Если в Кеше хранится не полный набор данных, то:
  - Если сортировка такая же, то:
    - Если фильтр точно такой же, то:
      - Если страница данных уже загружена, то отдать данные
      - Если страница за рамками загруженных данных, то загрузить данные с севера.
    - Если фильтр более точный, то :
      - Если в Кеше данные с первой позиции:
        - Провести фильтрацию имеющихся данных, и:
          - Если страница данных уже загружена, то отдать данные.
          - Если страница за рамками загруженных данных, то загрузить данные с севера.
        - Если в Кеше данные не с начала, то загрузить данные с сервера
    - Если фильтр менее точный, то перейти к загрузке данных с севера
  - Если сортировка другая, то загрузить данные с севера



- Если фильтр точно такой же, то:
  - Если сортировка такая же, то:
    - Если в кэше все данные, то отдать данные
    - Если данных в кэше достаточно, то отдать данные
    - Если данных не достаточно, обратиться на сервер
  - Если сортировка другая, то:
    - Если в кэше все данные, то провести сортировку и отдать данные
    - Если не все данные загружены, обратиться на сервер
- Если фильтр более точный:
  - Если в кэше все данные, отправить их на фильтрацию, сортировку и отдачу
  - Если в кэше часть данных, обратиться на сервер
- Если фильтр менее точный, то перейти к загрузке данных с севера
